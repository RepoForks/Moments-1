To be the best unofficial Instagram app
==========

## Features

- Material Design
- HTTP Proxy support

## Build

1. Create `app/client.properties`

   ```
   CLIENT_ID = "YOUR CLIENT ID HERE"
   CLIENT_SECRET = "YOU CLIENT SECRET HERE"
   ```

2. Build it

   ```shell
   ./gradlew build
   ```
